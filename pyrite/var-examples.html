Using url(), you can point to an external resource such as an image. You can also use it with raw data, which doesn't call any external resources.
 --background-image: url('https://leafyluigi.github.io/images/pexels-josh-sorenson-116359.png'); --home-icon: url('https://leafyluigi.github.io/images/pexels-chris-munnik-2604991.png');

?
Colors in CSS can be written in one of a few ways:
- HEX (displayed below)
- RGB (displayed at the bottom of page)
- HSL (Discord uses this for the saturation slider)
HEX is the most compact byte-wise, RGB is the most human-readable and HSL allows easy tinkering via calc().
The addition of an "A" at the end of the abbreviation or 4/8 digit long HEX allows for Alpha/Opacity.
None of the custom variables in this theme use HSL.

Note: --header-font isn't listed in the .theme.css file but uses the same fonts as --body-font by default. It can be manually added for customization.
Note: "Code" fonts are generally monospace/fixed-width.
Multiple font names within the same variable are for fallbacks and offer more compatibility for those without said font.
The default body font is the same as Discord's own with previously-used "Whitney" being before "gg sans".
--header-font is used in-place of Discord's --font-headline and for some headings.
--code-font is primarily used for codeblocks and sent txt files.
--body-font is otherwise used.
Click here for MDN page for @font-face at-rule.
Getting an external font:
You must know the name and source URL to use an external font.
Each @font-face at-rule always contain a font-family and a src descriptor.
These are used as the name of the font and the source URL, respectively.
The font-family is what you add to any of the below variables.
Other optional descriptors may be used such as font-style or font-weight. You can read up on these at the above link.
Some fonts can be added via a standard @import; though these usually have a @font-face within the file.

You can see my reasoning for the fonts here.
 --header-font: Whitney, "gg sans", "Noto Sans", "Helvetica Neue", Helvetica, sans-serif; --code-font: Hack, Monaco, Consolas, Courier, monospace; --body-font: Whitney, "gg sans", "Noto Sans", "Helvetica Neue", Helvetica, sans-serif;
